CIT 336 Schedule

The course is divided into four units. Each unit focuses on related topics, has a number of readings, learning activities and assessments. Because this is a web development class, all assessments are submitted by being posted to your web site as specified in this schedule or individual assessment descriptions.

Once posted, an assessment should be fully checked for operation and then remain unchanged until graded and the grade posted. Do NOT be late with assessments, late work is penalized 50%.

Lesson 04

This week and next week focus on using PHP with a database to develop a CRUD (Create, Read, Update and Delete) application to deal with content in a dynamic website. CRUD is an acronymn that represents: create (INSERT), read (SELECT), update and delete data in a database. These four interactions constitute the basic functionality of a dynamic web site.

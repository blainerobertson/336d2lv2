Acme Database Leveling

What?

In week 03 you created the acme database and imported a SQL file to create the inventory and categories tables. The clients table you created from scratch. The problem is that many of us are novices and may not have created the clients table completely or correctly.

This short activity is meant to fix that by making sure that all of our acme tables are the same ("leveled") before we begin using them in earnest.

Acme SQL File

What I didn't tell you earlier is that the code to create the clients table is already in the downloadable sql file that you used to create the other two tables, just commented out. We will uncomment the code and import the file again to make sure all of our acme databases are the same.

Prepare the File

  1. Download the Acme SQL file again.
  2. Unzip the file.
  3. Store the acme-db.sql file into your sql folder (created in week 03), for possible future use.
  4. Launch NetBeans.
  5. Find and open the acme-db.sql file.
  6. Scroll down to line 124 or so.
  7. Uncomment the following lines:
    • Lines 128 through 137
    • Lines 142 and 143
    • Lines 148 and 149

    To remove a comment, delete the -- on the left of the line.

  8. When done check that there are no errors (red icons or underlines) shown in the file!

Update the Database

  1. Launch XAMPP and start the Apache and MySQL servers.
  2. In a browser window, open the PHPMyAdmin tool.
  3. Click the acme database (on the left or in the "Databases" tab).
  4. Click the "Import" tab.
  5. Click the "Choose File" button.
  6. Navigate to the acme-db.sql file and click the "Open" button in the finder window.
  7. At the bottom of the Import window, click the "Go" button.
  8. If everything is successful you should see a green bar and on the left you should see the three tables listed beneath the acme database.
  9. That's it! Everyone who does this will have the exact same tables, table structures and data.
  10. Be sure to shut down your servers when you are done working with them.
